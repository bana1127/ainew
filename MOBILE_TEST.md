# ClubAgent 모바일 테스트 체크리스트

휴대폰 브라우저(iPhone Safari / Android Chrome)에서 다음 항목을 확인하세요.

## 접속 URL

```
https://agent.banawy.store
https://agent.banawy.store/dashboard
https://agent.banawy.store/assistant
https://agent.banawy.store/activities
https://agent.banawy.store/receipts
https://agent.banawy.store/payments
https://agent.banawy.store/notifications
```

## 체크리스트

### 레이아웃 기본

- [ ] 1. 외부 도메인(agent.banawy.store) 모바일 브라우저에서 접속 가능
- [ ] 2. 로그인 없이 Dashboard가 표시됨
- [ ] 3. 전체 페이지에서 가로 스크롤이 생기지 않음
- [ ] 4. 텍스트가 잘리거나 겹치지 않음

### 메뉴 / Sidebar

- [ ] 5. 모바일에서 좌측 상단 햄버거 메뉴(≡) 버튼이 보임
- [ ] 6. 햄버거 버튼 클릭 시 왼쪽에서 Sidebar Drawer가 열림
- [ ] 7. 메뉴 항목 클릭 시 해당 페이지로 이동 후 Drawer 닫힘
- [ ] 8. Drawer 바깥(오른쪽 어두운 영역) 클릭 시 닫힘
- [ ] 9. 활성 메뉴가 라벤더 색으로 하이라이트됨

### Dashboard

- [ ] 10. Hero 영역이 세로 배치로 표시됨

- [ ] 12. 운영 요약 카드 2열 배치

## 모바일에서 영수증 업로드 방법

1. `/assistant` 또는 `/receipts` 페이지 접속
2. 파일 업로드 영역 탭
3. iOS: "사진 앨범" 또는 "카메라" 선택  
   Android: "파일 관리" 또는 "카메라"에서 촬영 선택
4. 영수증 사진 선택
5. 분석 실행

---

## 모바일에서 Sidebar 사용법

1. 화면 좌측 상단 **≡** 버튼 탭
2. Drawer가 왼쪽에서 슬라이드 인
3. 원하는 메뉴 탭 → 페이지 이동 + Drawer 자동 닫힘
4. 또는 오른쪽 어두운 배경 탭으로 닫기
